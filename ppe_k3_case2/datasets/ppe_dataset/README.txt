PPE Synthetic Dataset Demo

Isi:
- 18 gambar train
- 6 gambar val
- 6 gambar test
- label YOLO untuk kelas:
  0 person
  1 helmet
  2 vest
  3 gloves
  4 safety_shoes
  5 goggles

Catatan penting:
Dataset ini adalah dataset sintetis / demo buatan otomatis.
Tujuannya untuk:
1. mengetes apakah training project bisa jalan,
2. mengecek struktur folder dataset,
3. menghasilkan model demo.

Dataset ini tidak cukup bagus untuk model PPE produksi nyata.
Untuk hasil akurat, tetap perlu dataset CCTV/factory asli dan jumlah jauh lebih banyak.
